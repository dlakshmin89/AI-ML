What happens if you use = or == in if loop
The difference between = and == is fundamental in programming.
	• = is an assignment operator — it assigns a value to a variable.
	• == is a comparison operator — it checks whether two values are equal.
In an if condition:
	• Using == is correct because we are checking a condition.
	• Using = usually causes an error (in most languages like Python and Java) because assignment is not allowed inside a condition.

In Python, using = inside an if condition gives a syntax error.
However, in some languages like C or C++, = inside an if does not give an error. Instead, it assigns the value and then evaluates it, which can cause logical bugs.
